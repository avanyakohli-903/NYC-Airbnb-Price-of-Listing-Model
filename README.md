# NYC-Airbnb-Price-of-Listing-Model

Project Overview: In this project, I created a decision tree model as well as a neural network to train on data including information about many airbnb listings. Information included many features such as bedroom count, location, host information, etc. The label and what I was trained the model to do is to predict whether a listing is high or low priced based on other information and features. 

Objectives and Goals: The goal was to create a model for clients to use to see what a listing may be priced as. High or low. 

Methodology: I used a decision tree model and neural network after cleaning the data using one hot encoding and adjusting the NaN values as well as removing unnecessary features. 

Results and key findings: The decision tree model happened to be more accurate than the neural network and it is easier to interpret as well. This is the model I would recommend to use for clients to get a better chance at recieving the right information from the model.

Visualizations: As seen within the file, for the decision tree, the visualization shows the highest accuracy with a hyperparameter value of 10 for the hyperparameter values I tested out. On my training curve, the training and validation loss stayed pretty stable throughout training. There is a small gap between the training and validation accuracy which could indicated slight overfitting.


Potential next steps: Next steps include gaining more training and testing data from more listings on Airbnb so that the model can have more data to learn on.
